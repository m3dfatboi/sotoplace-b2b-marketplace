### Core
- **Next.js 14** - React фреймворк с App Router
- **TypeScript** - полная типизация
- **Tailwind CSS v4** - utility-first стилизация

### UI & Components
- **shadcn/ui** - 13 компонентов (Button, Card, Table, Dialog, etc.)

### State Management
- **React Query** - серверное состояние, кеширование
- **Zustand** - клиентское состояние, персистентность

### API & Utils
- **Axios** - HTTP клиент с interceptors
- **date-fns** - работа с датами
- **clsx + tailwind-merge** - условные классы

## 🚀 Как запустить

### 1. Backend (если еще не запущен)

```bash
cd /Users/m3dv3d3v/sotoplace
docker-compose up -d
docker-compose exec backend alembic upgrade head
docker-compose exec backend python -m scripts.seed_data
```

Backend будет доступен на http://localhost:8000

### 2. Frontend

```bash
cd /Users/m3dv3d3v/sotoplace/frontend
npm install  # если еще не установлено
npm run dev
```

Frontend будет доступен на http://localhost:3000

### 3. Тестовые учетные данные

```
Admin:   admin@sotoplace.com / admin123
Manager: manager@sotoplace.com / manager123
Client:  client@sotoplace.com / client123
```
