# Frontend - Dashboard (Step 1) ✅

## Что создано

### Главная страница (Dashboard)
**Чистый, профессиональный B2B интерфейс**

#### Компоненты:
1. **Header** - навигация (Главная, Заказы, Каталог)
2. **KPI Cards** (4 карточки):
   - Всего заказов
   - Активные заказы (синий)
   - Выручка (зеленый)
   - Товары в каталоге
3. **Таблица заказов** (последние 5):
   - Номер заказа
   - Покупатель
   - Статус (цветные badges)
   - Сумма
   - Дата создания
4. **Быстрые действия**:
   - Создать заказ
   - Добавить товар

---

## Дизайн

**Стиль:** Минималистичный B2B
- Чистый белый фон
- Синяя палитра (доверие)
- Фокус на данных
- Без лишних украшений

**Цвета:**
- Primary: Blue-600 (#2563eb)
- Success: Green-600
- Background: Gray-50
- Cards: White с тенью

---

## Технологии

```
Next.js 14      - React фреймворк
TypeScript      - Типизация
Tailwind CSS    - Стилизация (чистая, без библиотек)
Axios           - HTTP клиент
date-fns        - Форматирование дат
```

---

## Как запустить

### 1. Backend (если не запущен)
```bash
docker-compose up -d
```

### 2. Frontend
```bash
cd frontend
npm run dev
```

Откройте: http://localhost:3000

---

## Интеграция с Backend

**API endpoints используются:**
- `GET /api/v1/orders` - список заказов
- `GET /api/v1/products` - список товаров

**Данные:**
- Реальные данные из backend
- Автоматический расчет статистики
- Форматирование дат на русском
