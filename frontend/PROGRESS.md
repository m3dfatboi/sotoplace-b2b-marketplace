# ✅ Frontend - 3 основные страницы готовы!

## Что создано

### Шаг 1: Главная страница (Dashboard) ✅
- 4 KPI карточки
- Таблица последних заказов
- Быстрые действия

### Шаг 2: Страница заказов ✅
- Поиск и фильтры
- Полная таблица заказов
- Статистика

### Шаг 3: Каталог товаров ✅
- Поиск по товарам
- Фильтр по статусу
- Grid с карточками товаров
- Статистика

---

## Технологии

```
Next.js 14      - React фреймворк
TypeScript      - Типизация
Tailwind CSS    - Чистая стилизация
Axios           - API клиент
date-fns        - Даты
```

---

## Как запустить

### Backend
```bash
docker-compose up -d
```

### Frontend
```bash
cd frontend
npm run dev
```

Откройте: http://localhost:3000

---

## Страницы

- **/** - Главная (Dashboard)
- **/orders** - Заказы
- **/products** - Каталог

---

## Интеграция с Backend

**Работающие endpoints:**
- `GET /api/v1/orders` - список заказов
- `GET /api/v1/products` - список товаров

**Данные:**
- Реальные данные из backend
- Автоматический расчет статистики
- Форматирование на русском

---

## Дизайн

**Стиль:** Минималистичный B2B
- Чистый белый фон
- Синяя палитра
- Фокус на данных
- Без лишних украшений

**Компоненты:**
- Cards (карточки)
- Tables (таблицы)
- Buttons (кнопки)
- Filters (фильтры)
- Badges (статусы)
