
### Цвета
```css
Основной текст: #1f1f1f (не #000000)
Вспомогательный: #a3a3a3 (не #737373)
Границы: #e5e5e5 (не #cccccc)
Акцент: #d18043 (furnita)
Фон страницы: #f5f5f5

```

```css

```
