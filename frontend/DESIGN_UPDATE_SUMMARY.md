## Принципы из крупных маркетплейсов

### 1. Информационная плотность (Ozon, WB)
- Ключевые метрики видны с первого взгляда
- Крупные числа, мелкий контекст
- Визуальные тренды (стрелки, проценты)

### 2. Визуальная иерархия (Яндекс Маркет)
- Важное - крупнее и контрастнее
- Вспомогательное - серым
- Группировка через spacing

### 3. Статусы и индикаторы
- Цветовое кодирование (красный/желтый/зеленый)
- Иконки для быстрого распознавания
- Бейджи для количественных показателей

### 4. Быстрые действия
- Кнопки всегда на виду
- Первичное действие выделено
- Вторичные - outline стиль

### 5. Адаптивность
- Сетка 4 колонки (desktop)
- Минимальная ширина виджета: 260px
- Responsive breakpoints готовы
